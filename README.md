## 포켓몬 도감
#### 사용한 기술
+ axios를 사용해서 포켓몬 데이터 추출
+ 파이어베이스를 이용해서 구글,깃허브, 이메일가입, 로그인 구현
+ 홈화면에 데이터정렬,검색필터, mui페이지네이션
+ 즐겨찾기 페이지 구현
+ firebase storage를 이용해서 개인 프로필 사진 올리기 구현
+ chart.js를 적용해서 데이터 시각화
+ react-router-dom 사용
+ styled components로 스타일링과 dark모드 구현
